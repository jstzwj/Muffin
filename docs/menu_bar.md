# Muffin 菜单栏实现说明

本文档记录当前 `MainWindow::setupMenuBar()` 的菜单结构、启用状态和已接入行为。M2 阶段的目标是先完成源码模式、文件会话和 Typora 风格外壳；未实现的 WYSIWYG、导出、主题等功能保留菜单入口但禁用。

## 启动参数

```powershell
Muffin.exe path\to\file.md
```

启动时传入一个文件路径会直接打开该 Markdown / 文本文档。打开成功后会写入最近文件列表。

## 段落(&P)

段落菜单已按 Typora 风格建立入口，包括 1-6 级标题、段落、公式块、代码块、引用、列表、任务列表、脚注、水平分割线、目录、YAML Front Matter 等。M2 阶段这些命令暂未接入 AST 编辑事务，因此均为占位禁用。

## 格式(&O)

格式菜单已建立加粗、斜体、下划线、代码、删除线、注释、超链接、图像、清除样式等入口。M2 阶段暂未实现结构化样式命令，因此均为占位禁用。

## 主题(&T)

主题菜单仅保留 Github、Newsprint、Night、Pixyll、Whitey 五个内置主题入口。

## 帮助(&H)

帮助菜单保留 Typora 风格文档和支持入口。目前仅“关于”可用，显示 Muffin 简介；其余入口暂禁用。

## 状态栏

状态栏当前显示：

- 左侧模式占位：源码模式 / 激活状态占位。
- 解析耗时：来自 `DocumentSession` 每次解析 Markdown 的耗时。
- 光标行列：来自 `SourceEditorWidget` 的 `QPlainTextEdit` 光标位置。
- 词数：当前源码文本的简单词数统计。

## 实现边界

- 文件读写当前统一使用 UTF-8。
- 最近文件使用 `QSettings` 保存，最多 10 个。
- 当前主编辑器仍是 M2 临时源码模式 `SourceEditorWidget`，不是最终 WYSIWYG 自绘视图。
- 未实现项必须保持 disabled，直到对应 controller / command / view 真正落地。
